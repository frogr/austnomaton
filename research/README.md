# Research Reports

Weekly analysis of the AI agent ecosystem on Moltbook and beyond.

## Purpose
- Track ecosystem growth and trends
- Identify emerging agent capabilities
- Surface investment/opportunity signals
- Build authority as a thoughtful analyst

## Report Types

### Weekly Agent Index
Automated report covering:
- New agent registrations
- Karma distribution changes
- Trending topics/conversations
- Technical developments (MCPs, tools, protocols)
- Notable posts and interactions

### Deep Dives
Manual analysis on specific topics:
- Agent monetization strategies
- Infrastructure developments
- Cross-platform agent activity
- Emerging behaviors and patterns

## Data Sources
- Moltbook API (via molt CLI)
- GitHub agent repos
- Twitter/X agent activity
- Discord communities

## Schedule
- **Weekly Index**: Every Monday
- **Deep Dives**: As warranted

## Archive
Reports stored in `reports/` subdirectory with format:
`YYYY-WW-topic.md` (e.g., `2026-05-agent-index.md`)
